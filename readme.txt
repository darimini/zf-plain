ZF-PLAIN
--------------------------------
библиотека на php для эмуляции
полёта самолёта по заданной 
траектории
--------------------------------

Описание проекта:
-----------------
Проект включает в себя саму библиотеку (/library/plain_library.php)
и UI для работы с ней 

за rEarth (радиус Земли) принято начение 6372795 м
для рассчёта расстояний используется формула гаверсинусов 

- getDistance() - высчитывает расстояние всего маршрута
- getTimeArrival(DateTime $data) - высчитывает
ориентировочное время прибытия в конечную точку
для указанной даты и времени.
- getPartDistance(int $n) - высчитывает
расстояние в километрах для каждого участка маршрута, 
где $n номер участка между точками в файле настроек
$n-1 и $n
- getPartTimeArrival(DateTime $data, $n) - высчитывает
ориентировочное время прибытия для указанной даты и времени
отправления в заданную точку, где $n - точка в файле настроек
- методы play() и getPlace() симулируют движение самолёта и выводят 
процент от пройденного маршрута соответственно; время симуляции 
совпадает с реальным временем

пример входного файла в корневом каталоге проекта (test.json)
{"tr":[[33,33],[37,24],[43,45],[60,64]],"speed":1100} 
- географические координаты и скорость
 

Системные требования:
----------------------
php 5.3
zend framework 1.12.x (для UI)
